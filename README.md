# LinkyDo

LinkyDo — веб-платформа для фрилансеров и самозанятых, предназначенная для:
- учета доходов,
- автоматического расчета налогов,
- формирования отчетов,
- управления заказами,
- анализа юнит-экономики.

## Технологии
- React
- Node.js
- Express
- MongoDB
- Stripe (test mode)
- ExcelJS

## Возможности MVP
- регистрация и авторизация
- публикация услуг
- оформление заказов
- тестовая оплата
- налоговый модуль
- генерация отчетов
- расчет юнит-экономики
